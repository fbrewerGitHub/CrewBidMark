Copy Legs to Clipboard

You can copy the legs for a trip that is selected in the Line Calendar.

To copy the legs for a trip to the clipboard:
  1. Select the desired trip in the Line Calendar.
  2. Select the Trip->Copy Legs to Clipboard menu item.

Once the Copy Legs to Clipboard menu item has been selected, the legs are copied to the clipboard in a format suitable for pasting into other programs, such as the FFDO schedule importer.

The Copy Legs to Clipboard menu item is enabled only when there is a trip selected in the Line Calendar.
