View Trip...

You may view any trip that is included with the bid data for the bid month, domicile, and crew position.

To view a trip:
  1. Select the Trip->View Trip... menu item to bring up the View Trip window and dialog.
  2. In the View Trip dialog, enter the desired Trip Number and Start Day, then click the OK Button.

Only trips that are part of of the original bid data are available for viewing. Partial trips and trips constructed by crew scheduling for pilot second round bids are not available. Partial trips may be viewed by entering the original pairing number.

If the Start Day text field is left blank or a number less than one is entered, the first day of the bid month will be used to display the trip. If a day greater than the number of days in the bid month is entered in the Start Day text field, the last day of the bid month is used.

The Trip window contains two buttons which allow you to view a different trip or copy legs for the selected trip to the clipboard.
