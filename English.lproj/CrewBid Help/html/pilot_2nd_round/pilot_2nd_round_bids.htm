Pilot Second Round Bids

Some trip data is not available for pilot second round bids. Specifically, those trips that are not part of the first round bid (i.e., those trips constructed for the second round) have only the following data available:

  • trip number,
  • departure and return times,
  • overnight cities,
  • trip length, and
  • trip pay.
To prevent inaccurate sorting, selection, and points assignments, a limited number of choices are available for pilot second round bids.
