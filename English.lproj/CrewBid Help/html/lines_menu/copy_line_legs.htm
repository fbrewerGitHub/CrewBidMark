Copy Legs to Clipboard

You can copy the legs for a line that is selected in the Lines Table.

To copy the legs for a line to the clipboard:
  1. Select the desired line in the Lines Table.
  2. Select the Lines->Copy Legs to Clipboard menu item.

Once the Copy Legs to Clipboard menu item has been selected, the legs are copied to the clipboard in a format suitable for pasting into other programs, such as the FFDO schedule importer.

The Copy Legs to Clipboard menu item is enabled only when there is a line selected in the Lines Table.
