Selecting Lines and Assigning Points

Commuting



The Commuting choices allow you to select and assign points to lines by:

  • the number of commutes to domicile required, and
  • the number of overnights in domicile required.

The number of commutes required for a line is determined by:

  • the number of trips in the line, and
  • whether back-to-back trips require an overnight in domicile.

The number of overnights in domicile for a line is determined by:

  • the start and end times for each day of the week, and
  • whether back-to-back trips require an overnight in domicile.

Normally, for commuting purposes, back-to-back trips should be considered to require an overnight in domicile. (A back-to-back trip is a trip that has either another trip that ends the day before the trip begins or another trip the begins the day after the trip ends.) If back-to-back trips require an overnight in domicile, the number of commutes required for a line will be the number of blocks of back-to-back trips. If back-to-back trips do not require an overnight in domicile, the number of commutes required for a line will be the number of trips.

Whether back-to-back trips are considered to require an overnight in domicile also affects the number of overnights in domicile for a line. If back-to-back trips require an overnight in domicile:

  • each pair of back-to-back trips require an overnight in domicile,
  • each trip that begins a block of back-to-back trips or is not back-to-back, and departs before the earliest depart time, requires an overnight in domicile, and
  • each trip that ends a block of back-to-back trips or is not back-to-back, and arrives after the latest arrive time, requires an overnight in domicile.

If back-to-back trips do not require an overnight in domicile, then an overnight in domicile will be required for each trip that departs before the earliest depart time, and an overnight in domicile will be required for each trip that arrives after the latest arrival time.


To select lines by number of commutes required:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Commuting item from the list on the left.
  3. Enter the maximum number of commutes required and select the checkbox for Selects lines with <number> or less commutes required.
To select lines by number of overnights in domicile:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Commuting item from the list on the left.
  3. Enter the maximum number of overnights in domicile and select the checkbox for Selects lines with <number> or less overnights in domicile.


To assign points for commutes required:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Commuting item from the list on the left.
  3. Enter the points to be assigned for each commute required and select the checkbox for Assign <number> points for each commute required.
To assign points for overnights in domicile:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Commuting item from the list on the left.
  3. Enter the points to be assigned for each overnight in domicile and select the checkbox for Assign <number> points for each overnight in domicile.
