Selecting Lines and Assigning Points

AM/PM



To select lines by AM/PM:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the AM/PM item from the list on the left.
  3. Select the checkbox that corresponds to AM or PM only.


To assign points to AM/PM lines:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the AM/PM item from the list on the left.
  3. Enter the desired points value.
  4. Select the corresponding checkbox.
