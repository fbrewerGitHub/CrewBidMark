Selecting Lines and Assigning Points

Aircraft Changes

Selecting lines and assigning points by aircraft changes are available for pilot bids, but are not available for flight attendant bids. Flight attendant bid data does not contain aircraft change information.



To select lines by aircraft changes:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Aircraft Changes item from the list on the left.
  3. Enter the maximum number of aircraft changes desired.
  4. Select the corresponding checkbox.


To assign points to aircraft changes:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Aircraft Changes item from the list on the left.
  3. Enter the desired points value.
  4. Select the corresponding checkbox.
