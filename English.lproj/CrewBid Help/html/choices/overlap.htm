Selecting Lines and Assigning Points

Overlap

Note: Selecting lines and assigning points for lines that have overlap is available for pilot first round bids only.




To select lines that do not have overlap:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Overlap item from the list on the left.
  3. Select the checkbox.


To assign points to lines by overlap:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Overlap item from the list on the left.
  3. Enter the desired points value.
  4. Select the corresponding checkbox.
