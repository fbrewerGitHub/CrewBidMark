Selecting Lines and Assigning Points

Trip Length



To select lines by trip length:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Trip Length item from the list on the left.
  3. Enter the maximum number of trips for each trip length.
  4. Select the corresponding checkbox for each trip length.


To assign points to trip length:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Trip Length item from the list on the left.
  3. Enter the desired points value for each trip length.
  4. Select the corresponding checkbox for each trip length.
