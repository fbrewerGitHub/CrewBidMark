Selecting Lines and Assigning Points

Deadheads

Note: The Deadheads option is not available for pilot second round bids. See the Pilot Second Round Bids topic for more information.




The Deadheads choices allow you to select and assign points to lines that have trips that begin with a deadhead to or end with a deadhead from a selected city. Such lines might be desirable for those crew members who commute from that city.

Note: Cities to which there are no deadheads at start in the bid period do not appear in the Deadhead at Start cities popup button; similarly, cities from which there is no deadhead at end do not appear in the Deadhead at End cities popup button.


To select lines by Deadhead at Start:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Deadheads item from the list on the left.
  3. Select the city to which you desire a deadhead in the Deadhead at Start popup button.
  4. Select the checkbox for Select lines with deadheads at start to: <selected city>.
To select lines by Deadhead at End:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Deadheads item from the list on the left.
  3. Select the city from which you desire a deadhead in the Deadhead at End popup button.
  4. Select the checkbox for Select lines with deadheads at end to: <selected city>.


To assign points by Deadheads at Start:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Deadheads item from the list on the left.
  3. Select the city to which you desire a deadhead in the Deadhead at Start popup button.
  4. Enter the desired points value and select the checkbox for Assign <number> points for each deadhead at start to <selected city>.
To assign points by Deadheads at End:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Deadheads item from the list on the left.
  3. Select the city from which you desire a deadhead in the Deadhead at End popup button.
  4. Enter the desired points value and select the checkbox for Assign <number> points for each deadhead at end from <selected city>.
