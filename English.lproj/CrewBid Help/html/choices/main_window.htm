Making Bid Choices

The Main Window

The Main Window opens when data for a bid period is loaded. In the Main Window, you may view line and trip data and make choices to create your ordered bid preference.

Lines Table

The Lines Table displays the lines in sorted order, along with information for attributes of each line. Clicking on a line will select the line and show its trips in the Lines Calendar.

Lines in the Lines Table may be manually sorted by dragging a line (or group of lines) to a new position.

The Lines Table columns may be moved by dragging a column to a new position. Columns may be added or removed in the Configure Tab of the Choices Tab View.

Line Calendar

The Line Calendar displays trips for the line that is selected in the Lines Table. Clicking on the trip text in the Line Calendar will select all days for that trip and show the trip in the Trip View.

Trip View

The Trip View displays the trip text for the trip selected in the Lines Calendar.

Choices Tab View

The Choices Tab View displays choices that control line selection, assigning points, and configuration of the Lines Table.
