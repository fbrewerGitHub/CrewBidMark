Selecting Lines and Assigning Points

Overnight Cities



Note: Cities for which there are no overnights in the bid period do not appear in the list of overnight cities.

To select lines by overnight cities:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Overnight Cities item from the list on the left.
  3. Select the checkbox(es) in the Sel column that corresponds to the city or cities where you desire or do not desire an overnight.
  4. Select the radio button that corresponds to at least one or no overnights.

You may quickly disable selection by overnight cities, without having to deselect each city, by selecting the Don't select by overnight cities radio button.



To assign points to overnight cities:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Overnight Cities item from the list on the left.
  3. Enter the desired points value in the Pts column that corresponds to the overnight city.
  4. Select the Use points for overnights checkbox.

You may quickly disable points for overnight cities, without having to zero the points value for each city, by deselecting the Use points for overnights checkbox.
