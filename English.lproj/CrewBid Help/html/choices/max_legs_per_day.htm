Selecting Lines and Assigning Points

Max Legs per Day

Note: The Max Legs per Day option is not available for pilot second round bids. See the Pilot Second Round Bids topic for more information.




To select lines by max legs per day:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Max Legs per Day item from the list on the left.
  3. Enter the maximum number of max legs per day desired.
  4. Select the corresponding checkbox.


To assign points by max legs per day:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Max Legs per Day item from the list on the left.
  3. Enter the desired points value and the maximum or minimum value for max legs per day.
  4. Select the corresponding checkbox.
