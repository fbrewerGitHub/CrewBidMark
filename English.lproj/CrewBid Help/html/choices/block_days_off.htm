Selecting Lines and Assigning Points

Block of Days Off



To select lines by block of days off:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Block of Days Off item from the list on the left.
  3. Enter the desired minimum block of consecutive days off.
  4. Select the checkbox.


To assign points to block of days off:
  1. Select the Select/Points tab item in the Choices Tab View.
  2. Select the Block of Days Off item from the list on the left.
  3. Enter the desired points value.
  4. Enter the minimum block of days off to which points will be assigned.
  5. Select the checkbox.
