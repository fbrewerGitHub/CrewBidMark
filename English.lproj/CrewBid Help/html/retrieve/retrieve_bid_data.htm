Retrieving Bid Data

CrewBid will automatically connect to the SWA server to download bid data.

To retrieve bid data for a new bid period:
  1. Select the File->New Bid... menu item to bring up the New Bid dialog.
  2. In the New Bid dialog, select the desired Month, Base, Position, and Round.
  3. Enter your SWALife User ID and Password, then click the OK button.

After connecting to the SWA server and downloading the bid data, CrewBid will display the new bid information in a new window.

When the New Bid dialog opens it will be populated with the appropriate Month, Base, Position, and Round. You should not have to change any of these values unless you have changed domicile or seat position.
