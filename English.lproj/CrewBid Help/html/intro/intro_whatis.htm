CrewBid Introduction

What is CrewBid?

CrewBid is an application that helps you prepare and submit bids. Using CrewBid, you can:

  • retrieve the month's line and trip data,
  • select and sort lines according to your priorities, and
  • submit the prioritized bid.

If you are new to bidding or have never used a bid sorting program, you should read the Bid Basics section. If you have experience with the bidding process, you can safely skip the Bid Basics section and start exploring the features of CrewBid.
